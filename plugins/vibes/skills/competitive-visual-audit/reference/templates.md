## Zig vs Zag Decision Matrix

```
ZIG FACTORS (Reasons to follow conventions):
□ Category requires trust/safety signals (healthcare, finance, legal)
□ Customers use visual conventions to identify legitimate options
□ Entering established market needing initial credibility
□ Differentiation comes from other factors (service, pricing, features)
□ Target audience values conformity/tradition

ZAG FACTORS (Reasons to break conventions):
□ Category is visually homogeneous (everyone looks the same)
□ Genuine philosophical/strategic difference to communicate
□ Target audience is tired of category sameness
□ Can sustain the difference with real substance
□ Willing to polarize some to attract others strongly
□ Brand philosophy challenges industry norms

SCORE:
Zig factors checked: ___
Zag factors checked: ___

RECOMMENDATION:
□ Zig (follow conventions) — because: _______________
□ Zag (break conventions) — because: _______________
□ Hybrid approach — zig on ___, zag on ___
```

## Output Validation Checklist

When reviewing competitive audit output, verify:

**Competitor Coverage**
- [ ] 5-10 competitors analyzed
- [ ] Direct and indirect competitors included
- [ ] Each has complete profile

**Visual Audit Matrices**
- [ ] Color audit matrix complete with hex values
- [ ] Typography audit matrix complete
- [ ] Imagery style audit complete
- [ ] Patterns and white space identified for each

**Strategic Analysis**
- [ ] Perceptual map with two relevant axes
- [ ] Competitors plotted on map
- [ ] White space identified
- [ ] Good/Different Chart with scores
- [ ] Quadrant analysis provided

**Differentiation Opportunities**
- [ ] Color opportunity identified
- [ ] Typography opportunity identified
- [ ] Imagery opportunity identified
- [ ] Zig vs Zag recommendation with rationale

**Evidence Quality**
- [ ] Specific examples from competitor sites
- [ ] User perception data (from reviews)
- [ ] Visual descriptions are specific (not "modern logo")
- [ ] Hex codes for colors

**Strategic Output**
- [ ] Only-ness Statement opportunity provided
- [ ] Clear recommendations prioritized
- [ ] Risk assessment included
