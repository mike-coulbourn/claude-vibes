## Testing Checklist

### 7-Day Iteration Plan

- [ ] Day 1: Practice pitches out loud, note awkward spots
- [ ] Day 2-3: Refine wording, time each version
- [ ] Day 4: Get feedback from 2-3 trusted people
- [ ] Day 5: Test in real conversations, track reactions
- [ ] Day 6: Review what worked, tweak CTA
- [ ] Day 7: Finalize and document backup variants

### Signs It's Ready

- [ ] Sounds natural, not scripted
- [ ] Hits time target comfortably
- [ ] Can deliver with eye contact
- [ ] Can answer follow-up questions smoothly
- [ ] Listener understands what you do
- [ ] Generates "tell me more" response
- [ ] Works with different audiences
- [ ] You can adapt it on the fly

### What to Track

- Number of follow-up conversations
- Questions asked (what do they want to know more about?)
- Parts that get the best reactions
- Parts that get confused looks

## Output Validation Checklist

Before finalizing pitch documentation, verify:

### Strategic Foundation
- [ ] Onlyness Statement completed and tested
- [ ] Primary framework selected with rationale
- [ ] Target audience clearly defined
- [ ] Pitch challenge documented (communicate, feel, sound, avoid)

### One-Liner
- [ ] Under 15 seconds when spoken
- [ ] Contains hook element
- [ ] Creates intrigue (not explanation)
- [ ] 2-3 alternative options provided
- [ ] Delivery notes included

### 30-Second Pitch
- [ ] 25-35 seconds when spoken at natural pace
- [ ] Follows 5-part structure (Hook, Problem, Solution, Differentiation, Close)
- [ ] S.T.A.R. moment identified
- [ ] What Is / What Could Be contrast clear
- [ ] Pause points marked
- [ ] Emphasis points noted
- [ ] 3 C's check passed (Clarity, Conciseness, Confidence)

### 60-Second Pitch
- [ ] 50-70 seconds when spoken
- [ ] Adds meaningful content beyond 30-second version
- [ ] Includes proof point or example
- [ ] Structure breakdown documented

### Founder Story Pitch
- [ ] Included if founder story adds credibility
- [ ] All 5 story elements present (Connection, Origin, Insight, Action, Vision)
- [ ] Weaves personal with business naturally

### Hooks
- [ ] 3+ hook options from different types
- [ ] Best context noted for each
- [ ] Hook selection by context documented

### CTAs
- [ ] Options for different contexts (meetings, demo, connection)
- [ ] Simple asks that require little from recipient

### Delivery
- [ ] Tone guidance provided
- [ ] Pacing notes included
- [ ] Strategic pause points marked
- [ ] Emphasis words identified

### Context Variations
- [ ] Networking events version
- [ ] Investor meetings version
- [ ] Customer conversations version
- [ ] Social situations version

### Follow-Up Preparation
- [ ] 3+ anticipated questions with responses
- [ ] Objection handling prepared
- [ ] "Tell me more" response ready

### Testing
- [ ] 7-day iteration plan documented
- [ ] Readiness checklist completed
- [ ] Tracking metrics defined

### Quality
- [ ] Document reviewed for AI patterns
- [ ] Language matches brand voice
- [ ] All pitches sound natural when spoken aloud
- [ ] No generic placeholder text remains
