# Golden Circle Purpose Templates

Structured templates for purpose discovery, validation, and documentation.

---

## Purpose Discovery Worksheet

Use this worksheet to guide the WHY discovery conversation.

```
### Purpose Discovery Worksheet

**Participant**: _______________
**Date**: _______________
**Facilitator/Partner**: _______________

---

#### Part 1: Peaks and Valleys

Draw a horizontal line. Plot significant life moments above (peaks) or below (valleys).

Key moments to explore:
- [ ] Childhood memories that shaped you
- [ ] Career highs and lows
- [ ] Relationships that changed you
- [ ] Moments when you felt most alive
- [ ] Moments when you felt most frustrated

---

#### Part 2: Story Prompts

Answer these for 10-12 specific stories:

1. **Formative Influence**: Who has helped make you the person you are today? What specifically did they do?

2. **Intrinsic Motivation**: Think of a day at work when you said "I would have done that for free." What happened?

3. **Values Revealed**: What was your worst day at work and why did it affect you so deeply?

4. **Early Joy**: What is your earliest specific happy childhood memory?

5. **Contribution Moment**: When have you felt you truly made a difference for someone?

6. **Defiance Moment**: When did you push back against how things "should" be done?

7. **Energy Source**: What activities make you lose track of time?

8. **Frustration Driver**: What injustice or problem keeps you up at night?

---

#### Part 3: Theme Identification

Themes appearing in 2+ stories:

| Theme | Stories Where It Appears | Strength (1-5) |
|-------|--------------------------|----------------|
| | | |
| | | |
| | | |
| | | |

**Golden Thread(s)** — themes that "shine brighter":
1. _______________
2. _______________

---

#### Part 4: WHY Statement Draft

**Format**: "To _____ so that _____"

First blank (contribution): _______________
Second blank (impact): _______________

**Draft WHY Statement**:
> _________________________________________________________________

**Does it feel true?** [ ] Yes [ ] Close but needs refinement [ ] Not quite
```

## Anti-Pattern Checklist

Use this checklist to ensure the purpose avoids common mistakes.

```
### Purpose Anti-Pattern Check

**Purpose Statement**:
> _______________________________________________

---

| Anti-Pattern | Status | Notes |
|--------------|--------|-------|
| **Starting with WHAT** — describes products first, never gets to why | [ ] Clear [ ] Risk | |
| **Confusing WHY with Profit** — "maximize value" language | [ ] Clear [ ] Risk | |
| **Inventing vs Discovering** — crafted without authentic connection | [ ] Clear [ ] Risk | |
| **Generic Purpose** — could apply to any company | [ ] Clear [ ] Risk | |
| **Purpose Washing Risk** — claims that can't be proven | [ ] Clear [ ] Risk | |
| **Losing the Customer** — so internal-focused it forgets audience | [ ] Clear [ ] Risk | |
| **Corporate Buzzwords** — leverage, synergize, best-in-class | [ ] Clear [ ] Risk | |

---

**Red Flag Words** (should NOT appear in purpose):
- [ ] "Best" / "excellence" / "quality"
- [ ] "Maximize" / "optimize" / "leverage"
- [ ] "Stakeholders" / "shareholders"
- [ ] "Solutions" / "services"
- [ ] "Industry-leading" / "world-class"

---

**Anti-Pattern Check Result:**
- [ ] **CLEAR** — No anti-patterns detected
- [ ] **RISKS IDENTIFIED** — Address: _______________
```

## Output Validation Checklist

When reviewing purpose documentation, verify:

**Discovery Quality**
- [ ] WHY Discovery process used (stories, not brainstorming)
- [ ] Themes identified from multiple sources
- [ ] Authentic connection to founder/organization story
- [ ] Emotional resonance present

**Statement Quality**
- [ ] Purpose uses "To _____ so that _____" format
- [ ] Under 15 words
- [ ] Jargon-free
- [ ] Couldn't be said by competitor
- [ ] Mission is action-oriented, present tense
- [ ] Vision is aspirational, future tense

**Validation Quality**
- [ ] Hedgehog three circles assessed
- [ ] Focus Lab framework applied
- [ ] Stress test passed
- [ ] Anti-patterns checked

**Documentation Quality**
- [ ] Full Golden Circle documented
- [ ] How to use each statement explained
- [ ] Connection to brand strategy noted
- [ ] Alternative formulations considered
