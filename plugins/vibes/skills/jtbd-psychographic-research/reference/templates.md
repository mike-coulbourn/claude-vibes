## Output Validation Checklist

When reviewing audience research output, verify:

**Job Coverage**
- [ ] Functional job clearly stated
- [ ] Emotional job captured (not just functional)
- [ ] Social job identified (how they want to be seen)

**Forces Analysis**
- [ ] Push forces documented (struggles)
- [ ] Pull forces documented (attractions)
- [ ] Anxiety forces documented (fears)
- [ ] Habit forces documented (comfort)
- [ ] Switching analysis provided

**Psychographic Depth**
- [ ] Limbic type identified with rationale
- [ ] VALS segment identified with rationale
- [ ] Values, motivations, and lifestyle captured

**Strategy Bridge**
- [ ] Research findings mapped to brand elements
- [ ] Implications for positioning stated
- [ ] Voice/tone recommendations provided

**Evidence Quality**
- [ ] Real quotes/examples included
- [ ] Sources cited
- [ ] Language patterns captured
