## Strategic Foundation Template

Use this to document the positioning inputs before generating taglines:

```markdown
## Strategic Foundation

### Brand Inputs

| Element | Content |
|---------|---------|
| **Brand Name** | [Name] |
| **Purpose (WHY)** | [Why the brand exists] |
| **Mission (HOW)** | [How they fulfill purpose] |
| **Vision (WHERE)** | [Where they're going] |
| **Core Values** | [3-5 values] |
| **Positioning** | [The territory claimed] |
| **Archetype** | [Primary archetype] |
| **Voice Traits** | [3-5 voice traits] |

### Onlyness Statement

Complete this positioning foundation:

"[Brand Name] is the only [type of company/product] that [unique offering] for [target audience] in [market] who want to [customer motivation] during a time of [cultural trend]."

### Brand Mantra

Three words that capture the brand's essence:

**[Emotional Modifier] + [Descriptive Modifier] + [Brand Function]**

Example: "Authentic Athletic Performance" (Nike)

### Trueline Development

The internal truth that the tagline will express publicly:

"[Brand Name] [helps/enables/empowers] [target audience] [achieve what outcome]"

### Audience Awareness Level

| Level | Description | Your Audience |
|-------|-------------|---------------|
| Most Aware | Knows product, ready to buy | [ ] |
| Product-Aware | Knows product, not convinced | [ ] |
| Solution-Aware | Knows solutions exist, not yours | [ ] |
| Problem-Aware | Has problem, doesn't know solutions | [ ] |
| Completely Unaware | Doesn't recognize the problem | [ ] |

**Implication for Tagline**: [How awareness level affects approach]

### The Tagline Challenge

Based on the above, the tagline must:

**Communicate:** [Core message in 1 sentence]
**Feel:** [Emotional quality in 2-3 words]
**Sound:** [Voice characteristics in 2-3 words]
**Avoid:** [What not to do]

### Competitive Tagline Landscape

| Competitor | Tagline | Type | Gap/Opportunity |
|------------|---------|------|-----------------|
| [Name] | "[Tagline]" | [Type] | [What's missing] |
| [Name] | "[Tagline]" | [Type] | [What's missing] |
| [Name] | "[Tagline]" | [Type] | [What's missing] |

**White Space Identified**: [Positioning opportunity competitors don't claim]
```

## Output Validation Checklist

Before finalizing tagline documentation, verify:

### Strategic Foundation
- [ ] Onlyness Statement completed and passes test
- [ ] Brand Mantra documented (3 words)
- [ ] Trueline distinct from tagline
- [ ] Audience awareness level identified with implications
- [ ] Competitive landscape analyzed

### Tagline Options
- [ ] 5-7 options generated from different angles
- [ ] Each option has type, device, word count documented
- [ ] Strategic rationale explains connection to brand strategy
- [ ] Strengths and considerations balanced
- [ ] At least one "wildcard" option explored

### Evaluation
- [ ] Evaluation matrix completed with scores
- [ ] AIDA scores for top 3 candidates
- [ ] ABC test completed
- [ ] Anti-pattern check run on all finalists
- [ ] Onlyness Test verified for top choices

### Recommendation
- [ ] Top choice identified with detailed rationale
- [ ] Runner-up noted with "when to use instead"
- [ ] Decision criteria documented

### Guidelines
- [ ] Logo/tagline relationship defined
- [ ] Context usage documented (web, social, print, etc.)
- [ ] Voice consistency reminders included

### Testing
- [ ] A/B test plan documented
- [ ] Pre-launch checklist completed
- [ ] Trademark/legal checked
- [ ] Post-launch metrics defined

### Quality
- [ ] Document reviewed for AI patterns
- [ ] Language matches brand voice
- [ ] All examples are concrete and specific
- [ ] No generic placeholder text remains
