## Output Validation Checklist

Before finalizing visual direction documentation, verify:

### Strategic Foundation
- [ ] Single-Minded Proposition completed
- [ ] Brand adjectives defined (3-5)
- [ ] Visual Word Translation completed for each adjective
- [ ] Archetype visual language documented
- [ ] Competitive visual landscape analyzed
- [ ] Visual white space opportunity identified

### Mood Board Direction
- [ ] Overall aesthetic described in 3 words + detailed description
- [ ] 3+ reference categories with rationale
- [ ] Textures and materials defined
- [ ] Emotional qualities documented (primary, secondary, anti-emotions)
- [ ] Strategic connection explained

### Logo Design Brief
- [ ] Single-minded proposition for logo defined
- [ ] Must communicate / Must NOT communicate listed
- [ ] Logo style approach recommended with rationale
- [ ] 2-3 conceptual directions explored
- [ ] Technical requirements documented
- [ ] Industry clichés and pitfalls identified
- [ ] Competitive logo landscape noted

### Photography Style
- [ ] Overall feel described
- [ ] Subject matter defined (primary, secondary, avoid)
- [ ] Composition guidance provided
- [ ] Lighting direction specified
- [ ] Color treatment detailed
- [ ] People direction (if applicable)
- [ ] Do's and Don'ts table completed
- [ ] Reference photographers/brands listed

### Visual System
- [ ] All 9 core elements addressed (as applicable)
- [ ] Hierarchy defined
- [ ] Flexibility guidelines (constant vs. variable)
- [ ] Application priorities listed
- [ ] Design principles checklist completed

### Quality
- [ ] Document reviewed for AI patterns
- [ ] Every visual choice has strategic rationale
- [ ] Language matches brand voice
- [ ] Specificity (not vague descriptions like "modern and clean")
- [ ] Actionable for a designer
